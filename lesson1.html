***...หน่วยที่ 1 การใช้คอมพิวเตอร์และระบบสารสนเทศเพื่องานอาชีพ...***

 1. การใช้คอมพิวเตอร์เบื้องต้น
 1.1 ความหมายของคอมพิวเตอร์
"คอมพิวเตอร์" (Computer) เป็นเครื่องจักรที่สามารถประมวลผลข้อมูล ได้โดยอัตโนมัติ ตามโปรแกรมที่มนุษย์ป้อนคำสั่งให้ทำ
ผลลัพธ์ที่ได้จากการประมวลผล คือ "สารสนเทศ" ซึ่งอาจแบ่งความหมายของคอมพิวเตอร์เป็น 2 ลักษณะ คือ ความหมายตามลักษณะทางกายภาพ และความหมายตามพจนุกรม
  1.1.1 ความหมายตามลักษณะทางกายภาพ เป็นเครื่องมือหรืออุปกรณ์อิเล็กทรอนิกส์สามารถอ่าน
และบันทึกข้อมูล โดยต้องอาศัยหน่วยความจำในการเก็บชุดคำสั่ง (Program) ไว้สำหรับควบคุมการทำงานของเครื่องคอมพิวเตอร์
  1.1.2 ความหมายตามพจนานุกรม หมายถึง เครื่องคำนวณ มีหน้าที่คำนวณและเปรียบเทียบหรือ
ประวลผลตามคำสั่งที่มนุษย์จัดเตรียมไว้ในรูปของโปรแกรมหรือชุดคำสั่งต่าง ๆ

 1.2 อุปกรณ์หลักของเครื่องคอมพิวเตอร์
   คอมพิวเตอร์โดยทั่วไปประกอบด้วยชิ้นส่วนต่าง ๆ มากมาย ส่วนประกอบหลัก ๆ คือ

1. เคส (CPU Case,System Unit) หัวใจหลักของเครื่องคอมพิวเตอร์ คือ หน่วยประมวลผลกลางหรือเรียกว่า ซีพียู ช่วยในการคิดคำนวณและอื่น ๆ
2. จอภาพ (Monitor) จอภาพแสดงผลการทำงาน เหมือนกับจอโทรทัศน์ แสดงอักษร ตัวเลข และภาพกราฟิกต่าง ๆ
ขนาดที่นิยมคือ 21 นิ้วขึ้นไป และมีความละเอียดระดับ Full HD 1080p เพื่อการใช้งานได้อย่างสะดวก
3. แป้นพิมพ์ (Keyboard) อุปกรณ์ป้อนคำสั่งของคอมพิวเตอร์ แป้นพิมพ์หรือคีย์บอร์ด มีปุ่มตัวอักษร ตัวเลขและโค้ดพิเศษเป็นอุปกรณ์หลักที่ใช้ในการนำข้อมูลหรือคำสั่งลงในเครื่องคอมพิวเตอร์ให้เครื่องคอมพิวเตอร์ทำงานต่าง ๆ แป้นพิมพ์โดยทั่วไปมี 101-108 คีย์
4. เมาส์ (Mouse) เป็นอุปกรณ์ที่ทำหน้าที่คล้ายกับมือของเรา ในการหยิบจับสิ่งต่าง ๆ ที่ปรากฏบนหน้าจอคอมพิวเตอร์
เมาส์มีหลายแบบ เช่น แบบลูกกลิ้ง แบบไร้สาย แบบออปติคัล เป็นต้น
5. ลำโพง (Speaker) ในปัจจุบันจำเป็นต้องใช้สำหรับมัลติมีเดีย ดูภาพยนตร์ ฟังเพลงหรือเล่นเกมหรือใช้ฟังบทเรียนผ่านคอมพิวเตอร์

 1.3 การประกอบเครื่องคอมพิวเตอร์
เมื่อซื้อเครื่องคอมพิวเตอร์ ผู้ขายจะแยกจอภาพ ตัวเครื่องคอมพิวเตอร์ คีย์บอร์ดและเมาส์ บรรจุลงในกล่องเพื่อให้
นำไปประกอบเองที่บ้าน การประกอบเครื่องคอมพิวเตอร์นั้นไม่ยากเลย ช่องเสียบสายต่าง ๆ ของเครื่องคอมพิวเตอร
์ออกแบบมาเฉพาะแต่ละส่วน รับรองต่อไม่ผิดแน่นอน

 1.4 การเชื่อมต่ออุปกรณืต่าง ๆ
เครื่องคอมพิวเตอร์สมัยใหม่มีการจัดวางตำแหน่งของพอร์ตต่าง ๆ คล้ายกับแบบเก่าในอดีต แต่พอร์ตบางชนิด
อาจจะไม่ได้ใช้งานแล้ว เช่น พอร์ตอนุกรม พอร์ตขนาน พอร์ตเกม บางเครื่องก็ลดหรือตัดพอร์ตพีเอสทู (PS/2) ออก และหันมาใช้งานพอร์ตแบบยูเอสบี (USB) แทน

 1.5 ข้อควรระวังในการใช้เครื่องคอมพิวเตอร์
1.5.1 ไม่ควรปิดเปิดเครื่องคอมพิวเตอร์ติดต่อกันทันที
1.5.2 ขณะใช้คอมพิวเตอร์ ไม่ควรนำอาหารและเครื่องดื่มเข้ามารับประทาน
1.5.3 ห้ามเคาะ เขย่า เหยียบ กระโดด บนโต๊ะเครื่องคอมพิวเตอร์
1.5.4 อย่าขีดเขียนสิ่งใดลงบนหน้าจอภาพ คีย์บอร์ด หรือส่วนใดส่วนหนึ่งของเครื่องคอมพิวเตอร์
1.5.5 เมื่อใช้คีย์บอร์ดต้องใช้อย่างเบามือ ไม่ควรกระแทกแรง ๆ
1.5.6 หลักจากใช้งานเสร็จ ควรคลุมเครื่องคอมพิวเตอร์ด้วยผ้าหรือพลาสติกป้องกันฝุ่น
1.5.7 อย่าให้อากศภายในห้องร้อนและชื้นมาก

 1.6 การแก้อาการเครื่องค้าง
   การทำงานกับคอมพิวเตอร์อาจพบปัญหาโปรแกรมที่กำลังใช้งานอยู่เกิด "อาการค้าง" ไม่สามารถใช้งานได้อาการแบบนี้
เรียกว่า "ระบบแฮงก์"
  วิธีแก้ไขอาการเครื่องแฮงก์สามารถทำได้ ดังนี้
1 ) กดปุ่ม Ctrl + Alt + Del บนแป้นพิมพ์
2 ) ปรากฏหน้าต่างให้เลือกตัวเลือก Task Manager
3 ) เลือกโปรแกรมที่ค้าง (Not responding) แล้วกดปุ่ม End task
(ถ้ายังแก้ไขไม่ได้ ต้องกดปุ่ม Reset เครื่องใหม่ หรือกดปุ่ม Power ที่เครื่องค้างไว้จนไฟดับ แล้วเปิดเครื่องใหม่)

 1.7 การปิดเครื่องอย่างถูกวิธี
   การปิดเครื่องให้ถูกวิธีเป็นเรื่องที่ไม่ควรละเลย เพราะขณะทำงาน ข้อมูลบางส่วนจะถูกโหลดไว้ในหน่วยความจำแรม
RAM เมื่อปิดเครื่อง ข้อมูลในหน่วยความจำแรม RAM จะหายไปทันทีโดยที่ไม่มีทางกู้คืนมาได้ ฉะนั้นก่อนปิดเครื่องจะต้องตรวจสอบเสมอว่าบันทึกไฟล์งานที่เปิดไว้เรียบร้อยหรือไม่
วิธีป้องกันการสูญหายของข้อมูลอย่างง่าย ๆ คือ
1 ) ทุกครั้งที่จะปิดเครื่อง ให้ออกจากโปรแกรมที่ทำงานอยู่ทั้งหมด แล้วจึงกดปุ่ม Start
2 ) ปิดเครื่องที่ปุ่ม Start > Power
3 ) คลิกที่ Shut down
