4.การจัดรูปแบบตารางสำเร็จรูป
   เมื่อสร้างตารางแล้ว การปรับแต่งหรือการจัดรูปแบบตารางสามารถกำหนดได้เอง โดยใช้แถบเครื่องมือออกแบบจัดรูปแบบตารางสำเร็จรูปได้สวยงานและรวดเร็ว ซึ่งสามารถทำได้ดังนี้
 4.1 จัดรูปแบบตารางสำเร็จรูป


1.คลิกเมาส์วางเคอร์เซอร์ที่เซลล์ใดก็ได้
2.คลิกแท็บ Design (ออกแบบ)
3.คลิกปุ่ม More (เพิ่มเติม)
4.ปรากฏลักษณะตาราง คลิกเลือกแบบที่ต้องการ
5.ได้ผลลัพธ์ตามรูปแบบตารางที่เลือก

 4.2 แปลงตารางกับข้อความ
   หากต้องทำงานกับข้อมูล (Data) หรือฐานข้อมูล (Database) หรือนำข้อมูลมาจากอินเทอร์เน็ตโดยข้อมูลอยู่ในตาราง ต้องการแปลงเอาข้อความในตารางออกมาเป็นตัวอักษรธรรมดา สามารถทำได้ตามขั้นตอนต่อไปนี้


1.เลือกตาราง หรือคลิกเมาส์วางเคอร์ซอร์เซลล์ใดก็ได้
2.คลิกแท็บ Layout (เค้าโครง)
3.คลิกที่รายการ Convert to text (แปลงเป็นข้อความ)
4.ปรากฏหน้าต่าง Convert Table to Text (การแปลงตารางเป็นข้อความ)
5.คลิกปุ่ม OK (ตกลง)
6.ตารางจะถูกเปลี่ยนเป็นข้อความธรรมดาเมื่อแปลงตารางเป็นข้อความเรียบร้อย ให้ทดลองเปลี่ยนจากข้อความเป็นตาราง
7.เลือกข้อความให้เป็นแถบสีเทา
8.คลิกแท็บ Insert (แทรก)
9.คลิกปุ่ม Table (ตาราง)


10.คลิกรายการ Convert Text to Table...(แปลงข้อความเป็นตาราง)
11.จะปรากฏหน้าต่าง Convert Text to Table(แปลงข้อความเป็นตาราง)
12.คลิกปุ่ม OK (ตกลง)
13.ข้อความจะเปลี่ยนเป็นตาราง

 

5.การพิมพ์เอกสารออกทางเครื่องพิมพ์
 5.1 การแสดงตัวอย่างก่อนพิมพ์
   การตรวจสอบเอกสารก่อนการพิมพ์เอกสารออกทางเครื่องพิมพ์ถือว่าเป็นเรื่องสำคัญก่อนการพิมพ์เอกสารออกมา ซึ่งมีวิธีการและกำหนดค่า ดังนี้


1.เปิดเอกสารที่ต้องการพิมพ์บนหน้าจอ
2.คลิกแท็บไฟล์ File (ไฟล์)
3.คลิกรายการ Print (พิมพ์)
4.ปรากฏหน้าต่าง Preview Pane ด้านขวาเพื่อแสดงตัวอย่างก่อนพิมพ์
5.เลือกลักษณะการแสดงตัวอย่างก่อนพิมพ์ในส่วนด้านล่างเลือกหน้า
6.คลิกเลือกปรับขนาดการแสดงเอกสาร

 5.2การพิมพ์เอกสารออกทางเครื่องพิมพ์
   เมื่อได้กำหนดรายละเอียดละแสดงตัวอย่างก่อนพิมพ์พร้อมที่จะพิมพ์เรียบร้อยแล้ว ต่อไปจะเป็นการกำหนดค่าเพื่อให้สั่งพิมพ์เอกสารออกทางเครื่องพิมพ์อย่างมีประสิทธิภาพ


1.คลิกแท็บไฟล์ (File)
2.คลิก Print (พิมพ์)
3.Copies (จำนวนสำเนา)คือจำนวนชุดที่จะสั่งพิมพ์
4.Printer (คลิกลูกศรเลือกเครื่องพิมพ์)
5.Printer Properties (ตั้งค่าเครื่องพิมพ์)
6.Settings (เลือกพิมพ์ทุกหน้า) Print All Pages หรือสามารถกำหนดค่าอื่น ๆ
7.Print One Sided (พิมพ์ทั้ง2ด้านของกระดาษเลือกพิมพ์ที่ละด้าน หรือพิมพ์ 2 ด้าน)
8.Collated (เลือกการเรียงหน้า กรณีพิมพ์หลายชุด)
9.Portrait Orientation (เลือกแนวการวางกระดาษ)
10.A4 (เลือกขนาดกระดาษ)
11.Normal Margins (เลือกระยะขอบ)
12.1Page Per Sheet (เลือกจำนวนหน้าต่อแผ่น
13.Page Setup (การตั้งค่าหน้ากระดาษ)
14.คลิกปุ่ม (พิมพ์) เพื่อสั่งพิมพ์เอกสารออกทางเครื่องพิมพ์
