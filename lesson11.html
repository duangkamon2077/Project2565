 2. กระบวนการทำงานของคอมพิวเตอร์
 2.1 หน่วยรับข้อมูล (Input Unit) ทำหน้าที่รับข้อมูลจากผู้ใช้ซึ่งผู้ใช้เป็นผู้ป้อนข้อมูลหรือคำสั่งส่งไปยังเครื่องคอมพิวเตอร์ ์เพื่อประมวลผลต่อไป อุปกรณ์รับข้อมูล ได้แก่ แป้นพิมพ์ (Keyboard) เมาส์ (Mouse) จอยสติ๊ก (Joystick)
สแกนเนอร์ (Scanner) เป้นต้น
 2.2 หน่วยความจำ (Memory) เป็นหน่วยความจำของเครื่องคอมพิวเตอร์ ทำหน้าที่เก็บข้อมูลหรือคำสั่งต่าง ๆ ที่รับเข้ามาไว้เพื่อประมวลผลและแสดงผล มี 2 ประเภท คือ
  2.2.1 หน่วยความจำหลัก (Main Memory) มี 2 ชนิด คือ
   1 ) หน่วยความจำหลักแบบอ่านได้อย่างเดียว เรียกว่า ROM เป็นหน่วยความจำแบบถาวรสำหรับเก็บคำสั่ง
   2 ) หน่วยความจำหลักแบบแก้ไขได้ เรียกว่า RAM เป็นหน่วยความจำแบบชั่วคราวสำหรับเก็บข้อมูลและชุดคำสั่ง เฉพาะเวลาที่มีกระแสไฟฟ้าหล่อเลี้ยงเท่านั้น
  2.2.2 หน่วยความจำสำรอง (Secondary Storage Unit) มีหน้าที่เก็บข้อมูลที่อยู่นอกเครื่อง เช่น อุปกรณ์เก็บข้อมูลสำรอง ฮาร์ดไดร์ฟ และแฟลชไดร์ฟ เป็นต้น
 2.3 หน่วยประมวลผลหรือหน่วยประมวลผลกลาง (Central Processing Unit : CPU) ทำหน้าที่นำข้อมูลที่ได้รับจากหน่วย
รับข้อมูลมาคำนวณหรือประมวลผลข้อมูล และส่งผลลัพธ์ไปเก็บไว้ในหน่วยความจำหลักประกอบด้วยส่วนสำคัญ 2 ส่วน คือ
  2.3.1 หน่วยควบคุม ทำหน้าที่ควบคุมการทำงานของอุปกรณ์ต่าง ๆ ในระบบทั้งหมด
  2.3.2 หน่วยคำนวณ ทำหน้าที่ประมวลผลข้อมูลทางคณิตศาสตร์เช่น บวก ลบ คูณ หาร
 2.4 หน่วยแสดงผล (Output Unit) เป็นหน่วยที่นำข้อมูลที่ได้รับการประมวลผลมาแสดงผลลัพธ์ที่ได้ผ่านทางจอภาพ หรือเครื่องพิมพ์ โดยอาจอยู่ในรูปของตัวเลข ตัวอักษร ภาพ แผนภูมิ ตาราง และเสียง

 3. เทคโนโลยีสารสนเทศ
 3.1 นิยามเกี่ยวกับเเทคโนโลยีสารสนเทศ
  ปัจจุบันเทคโนโลยีสารสนเทศมีบทบาทอย่างกว้างขวางในทุกวงการ และเทคโนโลยีสารสนเทศกลายเป็นเครื่องมือ
สำคัญของการทำงานทุกด้าน นับตั้งแต่ทางด้านการศึกษา พาณิชยกรรม เกษตรกรรม อุตสาหกรรม สาธารณสุข
การวิจัยและพัฒนา ตลอดจนด้านการเมืองและราชการ จะเห็นว่าไม่มีงานด้านใดที่ไม่มีผู้คิดประยุกต์หรือนำเทคโนโลยีสารสนเทศ
เข้าไปช่วยให้การทำงานนั้น ๆ มีประสิธิภาพและสิทธิผลยิ่งขึ้น

 3.2 ความหมายของเทคโนโลยี
  เทคโนโลยี (Technology) หมายถึง วิธีการปฏิบัติหรือดำเนินการใด ๆ ที่ใช้ความรู้ วิธีการหรือเทคนิคทางวิทยาศาสตร์ ์เพื่อช่วยให้การดำเนินการต่าง ๆ ในที่นี้ เทคโนโลยีจึงเป็นเครื่องมือที่ทำหน้าที่ในการจัดการสารสนเทศเพื่อส่งต่อไปยังบุคลากร
ที่ใช้ระบบสารสนเทศ องค์ประกอบของเทคโนโลยีคอมพิวเตอร์ประกอบด้วย Input (อินพุต) Processing (ประมวลผล) Output (เอาต์พุต) และ Memory (หน่วยความจำ)

 3.3 ความหมายของสารสนเทศ
  สารสนเทศ (Information) หมายถึง ข่าวสารที่ได้จากการนำข้อมูลดิบ (Raw Date) ไม่ว่าจะเป็นตัวหนังสือ ตัวเลข รูปภาพ
และอื่น ๆ มาคำนวณทางสถิติ หรือข้อมูลที่ผ่านการประมวลผลด้วยเครื่องคอมพิวเตอร์เพื่อให้ได้ผลลัพธ์ตามที่กำหนด
เรียกว่า สารสนเทศ และเป็นประโยชน์ต่อการดำเนินชีวิตของมนุษย์ เช่น เรียนรู้สภาพสังคมความเป็นอยู่ กฎเกณฑ์ และวิชาการ เป็นต้น

 3.4 ความหมายของเทคโนโลยีสารสนเทศ
  เทคโนโลยีสารสนเทศ (Information Technology : IT) หมายถึง เทคโนโลยีที่ใช้จัดการสารสนเทศเป็นเทคโนโลยี ีที่เกี่ยวข้องตั้งแต่การรวบรวม การจัดเก็บข้อมูล การประมวลผล การพิมพ์ การสร้างรายงาน การสื่อสารข้อมูล การนำเสนอสารสนเทศในรูปแบบต่าง ๆประกอบด้วยคอมพิวเตอร์ โทรคมนาคม และบุคลากร เป็นต้น
  เทคโนโลยีสารสนเทศ เกิดจาการนำทคโนโลยีสองสาขามาใช้ประโยชน์ร่วมกัน ได้แก่ เทคโนโลยีคอมพิวเตอร์และเทคโนโลยีการ
สื่อสารและโทรคมนาคม
  เทคโนโลยีสารสนเทศ ช่วยในการเก็บบันทึกข้อมูล ประมวลผลข้อมูล ตลอดจนการนำเสนอข้อมูลในรูปแบบต่าง ๆ
  เทคโนโลยีการสื่อสารและโทรคมนาคม ช่วยในการเผยแพร่ข้อมูลและข่าวสารต่าง ๆ
