***...หน่วยที่ 5 การสร้างตารางและการพิมพ์เอกสารออกทางเครื่องพิมพ์...***

1.ส่วนประกอบของตาราง
   ส่วนประกอบของตาราง แต่ละตารางจะประกอบไปด้วยส่วนประกอบต่าง ๆ ดังต่อไปนี้


1.Row (แถว) คือ แถวตามแนวนอน
2.Column (สดมภ์/คอลัมน์) คือ แถวตามแนวตั้ง
3.Cell (เซลล์) คือ ช่องภายในตาราง 1 ช่อง คือ 1 เซลล์

 

2.การใช้ตารางโดยใช้วิธีการต่าง ๆ
   เริ่มต้นสร้างตารางซึ่งสามารถสร้างตารางได้หลายวิธี ทำได้ดังนี้
  2.1 สร้างตารางโดยใช้เมนูคำสั่ง


1.คลิกแท็บ Insert (แทรก)
2.คลิกปุ่ม Table (แทรกตาราง)
3.คลิกรายการ Insert Table...(แทรกตาราง
4.กำหนด Table Size (ขนาดตาราง)
  Number of columns : (จำนวนคอมลัมน์)5
  Number of rows : (จำนวนแถว)3
5.คลิกปุ่ม OK (ตกลง)
6.แสดงตาราง 5 คอลัมน์ 3 แถว

  2.2 สร้างตารางโดยใช้ปุ่มเครื่องมือ


1.คลิกเมาส์วางเคอร์เซอร์ตำแหน่งที่ต้องการ คลิกปุ่ม Table (ตาราง)
2.เลื่อนเมาส์วางช่องตารางช่องแรก เลือกจำนวนตาราง 4 คอลัมน์ 3 แถว และคลิกเมาส์ช่องสุดท้าย
3.ได้ตาราง 4 คอลัมน์ 3 แถว

  2.3การใช้ตารางโดยใช้เครื่องมือวาดปุ่มตาราง


1.คลิกปุ่ม Table (ตาราง)
2.คลิกรายการ Draw Table (วาดตาราง) ตัวชี้เมาส์จะเป็นรูปดินสอ
 สร้างตาราง 3 แถว 3 คอลัมน์ ดังนี้
3.คลิกเมาส์แล้วลากบรรทัดแรก
4.ลากเส้นแบ่งคอมลัมน์ 2 คอลัมน์
5.ลากเส้นเพิ่มแถว 1 แถว
6.ลากเพิ่มอีก 1 แถว
7.จะได้ผลลัพธ์ ตาราง 3 คอลัมน์ 3 แถว
8.คลิกปุ่ม Draw Table (วาดตาราง) หรือคลิกปุ่ม Borders>Drae Table เพื่อยกเลิกการวาด

  2.4การสร้างตารางด้วยแม่แบบตาราง


1.คลิกแท็บ Insert (แทรก)
2.คลิกปุ่ม Table (ตาราง)
3.คลิกรายการ Quick Tables (ตารางด่วน)
4.ปรากฏรูปแบบเลื่อนแถบเลือกตารางที่ต้องการ เช่น ตารางคู่ สามารถแก้ไขข้อมูลได้
5.คลิกที่ Design (ออกแบบ)
6.เลือกการแรเงาสีพื้นหลัง Shading หรือเลือกเส้นขอบล่าง Border
